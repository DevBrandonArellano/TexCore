1.  **Registro de Empaques**: Crear registros de bultos o cajas vinculados a una **Orden de Producción** activa.
2.  **Cálculo de Pesaje**: Ingresar el Peso Bruto y Tara para que el sistema calcule el **Peso Neto**.
3.  **Generación de Etiquetas**: Imprimir etiquetas en formato **ZPL** para impresoras Zebra directamente desde el dashboard.
4.  **Selección de Contexto**: Asignar el lote producido a la máquina y turno correspondiente.
